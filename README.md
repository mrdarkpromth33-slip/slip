**PromptPay Payment System** - A complete self-hosted backend solution for processing PromptPay payments with QR code reading from bank transfer slips and webhook integration for Android notifications.

## ‚ú® Key Features

‚úÖ **PromptPay EMVCo Standard** - Generate compliant QR codes for Thailand payments  
‚úÖ **QR Code Reading** - Read and parse QR codes from slip images using pyzbar + OpenCV  
‚úÖ **Webhook Integration** - Receive payment notifications from Android apps  
‚úÖ **Micro-transaction Support** - Automatic decimal addition for accurate order matching  
‚úÖ **PostgreSQL Database** - Persistent transaction records with SQLAlchemy ORM  
‚úÖ **RESTful API** - Comprehensive endpoints with Swagger/ReDoc documentation  
‚úÖ **Docker Ready** - Production-ready containerization  
‚úÖ **Mobile Friendly** - Designed for integration with Android notification listeners
